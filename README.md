# pbtheme
Упаковщик/распаковщик конфигурационного файла (theme.cfg) темы для PocketBook (.pbt)

Поддерживается формат PocketBookTheme версии 1

SDK для сборки под PocketBook: https://github.com/pocketbook-free/SDK_481.git
